## 1. 产品概述

### 1.1 产品愿景

打造"发票界的Markdown编辑器"——30秒完成一张专业发票，让自由职业者和小微企业主告别复杂财务软件。

### 1.2 目标用户

| 用户类型 | 占比 | 核心需求 | 使用场景 |
|---------|------|---------|---------|
| 自由职业者 | 40% | 快速开票、专业形象 | 项目收款、服务收费 |
| 小微企业主 | 35% | 批量管理、数据统计 | 日常经营、财务记录 |
| 个体工商户 | 25% | 简单易用、移动开票 | 现场收费、移动办公 |

### 2.2 P0 功能列表（MVP必须包含）

#### F01 发票创建与编辑

| 功能ID | F01 |
|-------|-----|
| **功能名称** | 发票创建与编辑 |
| **优先级** | P0 |
| **描述** | 核心功能，支持创建、编辑、删除发票 |

**功能详情**:

1. **发票信息录入**
   - 发票编号（自动生成/自定义）
   - 开票日期（默认当天）
   - 到期日期（可选）
   - 发票标题/备注

2. **销售方信息**
   - 公司名称
   - 联系人
   - 地址
   - 电话
   - 邮箱
   - 银行账户信息
   - Logo上传

3. **购买方信息**
   - 客户名称
   - 联系人
   - 地址
   - 电话
   - 邮箱

4. **商品/服务明细**
   - 项目名称
   - 描述
   - 数量
   - 单价
   - 小计（自动计算）
   - 支持添加多行

5. **金额计算**
   - 小计（自动求和）
   - 税率设置（可配置）
   - 税额（自动计算）
   - 折扣（可选）
   - 总计

6. **其他功能**
   - 保存草稿
   - 复制发票
   - 删除发票
   - 发票预览

**验收标准**:
- [ ] 用户可在30秒内完成一张基础发票的创建
- [ ] 所有字段可正确保存和编辑
- [ ] 金额计算准确无误
- [ ] 支持中英文输入
